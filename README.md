 QRGen – QR Code Generator

A modern QR code generator built using React, TypeScript, and Tailwind CSS.

✨ Features

- ⚡ Real-time QR code generation  
- 📥 Download QR as PNG  
- 🎨 Clean and responsive UI  
- 🌙 Dark mode support  
 🛠️ Tech Stack

- React + Vite  
- TypeScript  
- Tailwind CSS  
- react-qr-code
    
  🌐 Live Demo
https://qrgen-inky.vercel.app/

 📦 Installation

```bash
git clone https://github.com/Mrkryzer/qrgen.git
cd qrgen
